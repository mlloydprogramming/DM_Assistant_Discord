# dm_assistant_bot
This is a DM assistant bot being created for the sole purpose of helping run a rogue-lite, westmarch style dnd adventure

## How to run

To run this I've set everything up using uv. I believe you would run this by running `uv run main.py` ohh and you need to configure a discord app with user scope `applications.commands` and Guild Install Scopes `applications.commands` and `bot` the the following permissions

```
Attach Files
Embeded Links
Read Message History
Send Messages
Use Slash Commands
View Channels
```
